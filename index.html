HOW IT WORKS

x402 Payment + ENS Identity

Payment (x402 on TON)

1. Call a DVM → HTTP 402
2. User approves or agent signs
3. USDT Jetton transfer on TON
4. Gateway verifies → DVM executes
5. Owner earns, caller gets data

Identity (ENS on Sepolia)

Every DVM gets an on-chain name:
weather.uqa6ew1b.vendton.eth
Text records: address.ton + description
Agents discover DVMs by ENS name
Identity on ETH · Payments on TON
