How It Works

x402 on TON

HTTP 402 → sign USDT Jetton → verify on-chain → execute

ENS on Sepolia

On-chain discovery: agents find DVMs via ENS or gateway API

Serverless DVMs

JS code in vm sandbox, 10s timeout, fetch enabled

TON Connect

Users approve payments from their own wallet
